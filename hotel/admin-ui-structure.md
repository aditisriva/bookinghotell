# BookHotel Admin UI Structure

## Page hierarchy
1. Admin Dashboard
   - Hero overview
   - KPI cards
   - Revenue and booking charts
   - Core admin screens grid
   - Recent activities
   - Pending approvals
2. User Management
   - Search and filter toolbar
   - User table with actions
   - Details and moderation modal
3. Hotel Manager Management
   - Manager listing and approval controls
   - Suspend and details actions
4. Hotel Management
   - Hotel list and approval workflow
   - Edit and status management actions
5. Booking Management
   - Reservation table with search and filters
   - Booking detail and status update modal
6. Review Management
   - Review moderation and reporting actions
7. Reports & Analytics
   - Revenue, booking, growth, and hotel performance charts
8. System Settings
   - Website settings, contact details, policies, and platform configuration
9. Admin Profile
   - Profile details, password change, and security settings

## Component hierarchy
- Sidebar navigation
- Top bar with search, notifications, and profile menu
- Page header and breadcrumbs
- Statistic cards
- Analytics cards and charts
- Tables with filters and pagination-ready layout
- Modals for approvals, editing, and status updates
- Empty/loading-state placeholders

## Visual system
- Bootstrap 5 layout
- White base with blue primary accents
- Rounded cards and modern spacing
- Responsive behavior for desktop, tablet, and mobile
- Clean icons, professional tables, and SaaS-style surfaces
